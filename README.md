# ImageEditingApp

An image editing app that uses AI API integration alongside a programmatic interface.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

Enter your OpenAI API key (key icon in the app) to use AI features: generate from text, remove background, edit with prompt, upscale.
